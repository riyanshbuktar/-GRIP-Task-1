# GRIP Task 1: Student Score Prediction


This project was completed as part of The Sparks Foundation GRIP internship Task 1. The goal of this project is to predict a student's percentage score based on the number of hours they studied. I used Python and supervised machine learning techniques to train regression models and compare their performance. The notebook includes steps such as importing the dataset, exploring the data, visualizing the relationship between study hours and scores, training machine learning models, and evaluating the predictions. This project helped me practice data analysis, regression modeling, model evaluation, and using Python libraries for machine learning.

## Project Objective


The main objective of this project is to build a machine learning model that can predict student scores using study hours as the input feature. For example, if a student studies for a certain number of hours, the model should estimate the percentage score they may receive.

## Tools and Technologies Used


- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Linear Regression
- Ridge, Lasso, and ElasticNet Regression
- Random Forest Regressor

## What the Project Does

This project takes a small dataset containing student study hours and scores. It analyzes the relationship between the two variables and then trains supervised learning models to make predictions. The project compares different regression models to see which one performs better. It also uses evaluation metrics such as Mean Absolute Error and Mean Squared Error to measure the model’s accuracy.

## Steps Followed


1. Imported the required Python libraries.
2. Loaded and displayed the student study hours dataset.
3. Explored the data to understand the relationship between hours studied and scores.
4. Split the dataset into training and testing data.
5. Trained supervised machine learning models.
6. Compared model performance using error metrics.
7. Used the model to predict student scores based on study hours.

## Challenges and Bugs Fixed


One challenge in this project was understanding how to properly train and test machine learning models using a small dataset. I also had to make sure the input data was shaped correctly before passing it into the model. Another challenge was comparing multiple regression models and understanding which evaluation metric gave the best idea of model performance. By testing different models and reviewing the output, I improved my understanding of regression and supervised learning.
